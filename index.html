  1. Concepto de Internet

    Internet es una red mundial de computadoras interconectadas que se comunican entre sí mediante un conjunto de protocolos comunes. Su objetivo principal es el intercambio de información.

    • Origen: Surge en los años 60 como proyecto militar (ARPANET).
    • Evolucion:
    1. Década de 1980: conexión entre universidades.
    2. Década de 1990: expansión global y aparición de la Web.
    3. Actualidad: acceso ubicuo mediante dispositivos móviles.

    • Características principales:

    • Global y descentralizada.
    • Basada en protocolos TCP/IP.
    • Permite la comunicación y el intercambio de datos en tiempo real.

  2. Principales protocolos

    • Transmisión de ficheros:

    1. FTP (File Transfer Protocol)
    2. SFTP (Secure File Transfer Protocol)

    • Correo electrónico:

    1. IMAP
    2. POP3
    3. SMTP

    • Navegación web:

    1. HTTP (HyperText Transfer Protocol)
    2. HTTPS (HTTP Secure)

    • Otros protocolos:

    • DNS (Domain Name System)
    • SSH (Secure Shell)
    • Telnet (obsoleto)

  3. Formas de conectarse

    1. Red Telefónica Conmutada (RTC): La línea telefónica tradicional, requiere un módem.
    1. RDSI (Red Digital de Servicios Integrados): Permite voz y datos simultáneamente.
    2. ADSL (Línea de Abonado Digital Asimétrica): Utiliza la línea telefónica de cobre pero con una frecuencia diferente.
    3. Fibra óptica:
      • Gran ancho de banda y velocidad estable.
      • Utiliza pulsos de luz en lugar de corriente eléctrica.
      • Puede llegar hasta el hogar (FTTH) o combinarse con coaxial (HFC).
    4. Conexión móvil:
      1. 5G — Alta velocidad y baja latencia.
      2. 4G LTE
      3. 3G
    5. Conexión por satélite: Ideal para zonas rurales o de difícil acceso terrestre.

  4. Servicios más utilizados

    A. World Wide Web (WWW):

    • Navegación mediante navegadores (Chrome, Firefox, Edge).
    • Lenguajes principales: HTML, CSS, JavaScript.

    B. Correo electrónico:

    • Servicios: Gmail, Outlook, ProtonMail.
    • Ventajas:
      1. Comunicación rápida y gratuita.
      2. Permite adjuntar archivos.

    C. Redes sociales:

    1. Facebook
    2. Instagram
    3. X (Twitter)

    D. Servicios en la nube:

    • Google Drive
    • Dropbox
    • OneDrive

    1. Seguridad en la red

      • Buenas prácticas:

      1. Usar contraseñas seguras.
      2. No compartir datos personales en sitios no confiables.
      3. Mantener el software actualizado.

        • Amenazas comunes:

        1. Phishing (suplantación de identidad).
        2. Malware y virus.
        3. Ransomware.

          • Medidas preventivas:

          1. Instalar antivirus.
          2. Usar VPN.
          3. Utilizar certificados HTTPS.
